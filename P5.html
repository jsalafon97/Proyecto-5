2.1 - Funciones a implementar¶

A continuacin se especificarn las funciones auxiliares a desarrollar, para la construccin de la cadena de Markov:

  1. energia_diaria(archivo_json): Importa la base de datos completa en formato JSON y calcula la energa diaria usando la regla del trapecio y retorna un vector con el valor de energa de cada da.

  2. definicion_estados(vector_energia, numero_estados): Clasifica a cada valor de energa en el rango de 1 a numero_estados segn el nivel de energa y retorna un vector con cada estado.

  3. probabilidad_transicion(vector_estados, numero_estados, presente, futuro): Calcula la probabilidad de transicin entre un estado inicial $i$ en $t$ y un estado futuro $j$ en $t+1$. Retorna la probabilidad $\Pi_{i,j}$ de transicin entre $i$ y $j$, donde:

\begin{equation} P(X_{t+1} = j \mid X_{t} = i) = \Pi_{i,j} \end{equation}

import pandas as pd
import numpy as np
from datetime import datetime

def energia_diaria(archivo_json):
    '''Importa la base de datos completa y devuelve
    un vector con la energa diaria, en MWh.
    
    :param archivo_json: el contenedor con datos crudos
    :return: el vector de energa diaria
    '''
    # Cargar el "DataFrame"
    df = pd.read_json(archivo_json) 

    # Convertir en un array de NumPy
    datos = np.array(df)  

    # Crear vector con todos los valores horarios de demanda
    demanda = []

    # Extraer la magnitud de la demanda para todas las horas
    for hora in range(len(datos)):
        demanda.append(datos[hora][0]['MW'])

    # Separar las magnitudes en grupos de 24 (24 h)
    demanda = np.split(np.array(demanda), len(demanda) / 24)

    # Crear vector para almacenar la energa a partir de la demanda
    energia = []

    # Calcular la energa diaria por la regla del trapecio
    for dia in range(len(demanda)):
        E = round(np.trapz(demanda[dia]), 2)
        energia.append(E)

    return energia

import numpy as np

def definicion_estados(vector_energia, estados):
    '''Una funcin que se encarga de retornar
    los lmites del rango de energa para
    una cantidad arbitraria de estados sobre 
    la base del vector de energa.
    
    :param energia: vector de energa diaria
    :param estados: el nmero de estados
    :return: el vector de estados
    '''
    
    minimo = np.min(vector_energia)
    maximo = np.max(vector_energia)
    segmento = (maximo - minimo)/estados
    vector_estados = np.empty(len(vector_energia))
    
    for i, dia in enumerate(vector_energia):
        diferencia = dia - minimo
        proporcion = diferencia // segmento
        vector_estados[i] = proporcion + 1
        
    return vector_estados

2.1.3 - Calculando la ocurrencia de las transiciones por estado con calcular_transiciones¶

El objetivo de la funcin ser retornar el nmero de ocurrencias (y por tanto la frecuencia relativa) de la transicin de un estado presente $i$ a un estado prximo $j$; es decir, retorna puntualmente una probabilidad $\Pi_{ij}$ de transicin entre las muchas que puede contener la matriz de transicin $\Pi$ en funcin de sus $N$ estados. Obsrvese la siguiente matriz de estados generalizada:

$$ \Pi = \begin{bmatrix} \Pi_{11} & \ldots & \Pi_{1N} \\ \Pi_{21}& \ldots & \Pi_{2N}\\ \vdots& \ddots & \vdots\\ \Pi_{N1} & \ldots & \Pi_{NN} \end{bmatrix} $$

Puesto que $\Pi$ siempre es una matriz cuadrada, entonces habr $N^2$ probabilidades de transicin dentro de la misma. Ahora, dado a que la funcin calcular_transiciones retorna solo una de estas probabilidades, por ejecucin.

Nota: Qu ocurrira si se analiza un proceso con $N=10$ estados? Evidentemente, no sera prctico ejecutar esta funcin $10^2$ veces para completar la matriz $\Pi$.

import numpy as np

def probabilidad_transicion(vector_estados, numero_estados, presente, futuro):
    '''Una funcin que se encarga de calcular
    la probabilidad de ocurrencia de la transicin
    entre un estado inicial 'i' y un estado futuro 'j'.
    
    :param vector_estados: el vector con los todos los estados
    :param presente: el nmero del estado presente
    :param futuro: el nmero del estado futuro
    :return: la probabilidad de transicin
    '''
    
    # Recorrer el vector_estados
    ocurrencias_i = 0
    ocurrencias_i_j = 0
    for i, estado in enumerate(vector_estados[0:-1]):
        if estado == presente:
            ocurrencias_i += 1
            if vector_estados[i+1] == futuro:
                ocurrencias_i_j += 1
    
    # Clculo de la probabilidad
    probabilidad = ocurrencias_i_j / ocurrencias_i
    
    return probabilidad

from numpy import random
from scipy import stats

def parametros_asignados(digitos):
    '''Elige un valor t aleatoriamente,
    dos estados arbitrarios i y j
    '''
    
    random.seed(digitos)
    estados = [i+1 for i in range(10)]
    T = stats.expon(2)
    t = int(T.rvs())
    i = estados[random.randint(0, len(estados))]
    j = estados[random.randint(0, len(estados))]
    print('t: {}, i: {}, j: {}'.format(t, i, j))
    return t, i, j
